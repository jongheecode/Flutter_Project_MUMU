# [Music Mood] : Flutter & Node.js 기반 하이브리드 스트리밍 플랫폼
## AI 협업 생산성 극대화 및 구조적 리팩토링

### 개요: Flutter와 Node.js를 활용한 음악 감상 및 사용자 맞춤형 플랫폼

프로젝트 과정: 
1. AI 기반 프로토타이핑: Gemini를 활용해 초기 아키텍처 및 핵심 로직 초안 구축
2. 리팩토링: AI가 생성한 코드의 기술적 한계를 분석하고, 가독성과 유지보수성을 위해 직접 폴더 구조(MVVM) 및 상태 관리 로직 개선
3. 풀스택 구현: Node.js와 MySQL을 직접 연동하여 데이터 영속성 관리 및 API 설계
